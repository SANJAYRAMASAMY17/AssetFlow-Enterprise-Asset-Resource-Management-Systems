# AssetFlow - Asset Management System

AssetFlow is a comprehensive web application for managing organization assets, employees, departments, asset allocations, transfers, bookings, maintenance, and audits.

## Features
- Organization Management (Departments, Categories, Employees)
- Asset Inventory Tracking
- Asset Allocations (Assignments and Returns)
- Inter-Department Transfers
- Resource Bookings
- Maintenance Logs
- Physical Audits
- Activity Timeline & Notifications
- Reports & Analytics

## Tech Stack
- Frontend: React 18, Vite, Tailwind CSS, Recharts, TanStack Query, React Router
- Backend: Node.js, Express, Prisma ORM
- Database: PostgreSQL
